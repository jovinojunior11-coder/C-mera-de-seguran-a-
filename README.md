# C-mera-de-seguran-a-
Esta ferramenta web transforma celulares antigos em câmeras de segurança via navegador. Sem instalar apps, ela transmite vídeo e áudio em tempo real entre dois dispositivos usando tecnologia P2P (WebRTC). Basta abrir o link, definir um como "Câmera" e outro como "Monitor". Ideal para babá eletrônica ou vigiar pets com total privacidade.
